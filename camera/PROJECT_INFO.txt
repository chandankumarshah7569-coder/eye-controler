================================================================================
                        EYE CONTROLLER PROJECT
                           Version 1.0.0
================================================================================

PROJECT NAME: Unified Eye Controller
DESCRIPTION: Control videos and mouse with your eyes using AI
MAIN FILE: unified_eye_controller.py
LANGUAGE: Python 3.8+

================================================================================
                              FEATURES
================================================================================

MODE 1: VIDEO SCROLL MODE
- Blink to scroll through reels/shorts
- Wink for likes and pause/play
- Perfect for YouTube Shorts, Instagram Reels, TikTok

MODE 2: MOUSE CONTROL MODE
- Eye gaze controls cursor
- Blink for clicks
- Hand gestures for scrolling

================================================================================
                           QUICK START
================================================================================

WINDOWS:
  Double-click: run.bat

MAC/LINUX:
  chmod +x run.sh
  ./run.sh

MANUAL:
  pip install -r requirements.txt
  python unified_eye_controller.py
