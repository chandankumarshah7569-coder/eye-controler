# Quick Start Guide

Get up and running with Eye Controller in 3 minutes!

## 🚀 Installation

### Windows

1. **Double-click `run.bat`** - That's it! The script will:
   - Check if Python is installed
   - Install dependencies automatically
   - Launch the application

### Mac/Linux

1. **Make the script executable:**
   ```bash
   chmod +x run.sh
   ```

2. **Run the script:**
   ```bash
   ./run.sh
   ```

### Manual Installation

```bash
# Install dependencies
pip install -r requirements.txt

# Run the application
python unified_eye_controller.py
```

## 🎮 First Time Use

### Step 1: Choose Your Mode
When the app opens, you'll see two buttons at the top:
- **📱 Video Scroll** - For watching videos hands-free
- **🖱️ Mouse Control** - For controlling your computer

### Step 2: Start
1. Click the mode you want
2. Click **▶️ START**
3. Position yourself in front of the webcam

### Step 3: Try It Out

**Video Scroll Mode:**
- Hold a blink for 0.3 seconds → Scroll down
- Blink twice quickly → Scroll up

**Mouse Control Mode:**
- Look around → Cursor moves
- Single blink → Left click
- Double blink → Right click

## 💡 Quick Tips

1. **Good Lighting** - Make sure your face is well-lit
2. **Camera Position** - Place camera at eye level
3. **Distance** - Sit 1-2 feet from the camera
4. **Practice** - Takes 2-5 minutes to get comfortable

## 🐛 Common Issues

**Webcam not opening?**
- Close other apps using the camera
- Check camera permissions

**Not detecting blinks?**
- Improve lighting
- Blink more deliberately

**Cursor too jittery? (Mouse mode)**
- Keep your head still
- Adjust smoothing in settings
